# Техническое задание: Разработка Task Manager (MVP)

## Стек технологий
- **Фреймворк:** Spring Boot
- **Система сборки:** Maven
- **JDK:** 25
- **База данных:** PostgreSQL (развёртывание в Docker-контейнере)

## Описание задачи
Необходимо разработать серверное приложение типа Task Manager с реализацией
CRUD-операций. На данном этапе авторизация не требуется.
Пользовательский интерфейс (UI) не предусмотрен.

## Модели данных
- `User` — пользователь системы
- `Task` — задача
- `Category` — категория задачи (примеры значений: `Home`, `Work`, `Education` и др.)

## Функциональные требования
- Полный набор CRUD-операций для всех сущностей
- Реализация фильтрации и выборки задач по категориям
- Интеграция Swagger (OpenAPI) для документирования REST API

## Тестирование
Проверка работоспособности приложения осуществляется посредством Postman.

## Примечание
Функциональность приложения будет расширяться в последующих итерациях.
